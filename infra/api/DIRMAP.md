| --- | --- |

| Directory | Summary |
| --- | --- |
| src | The src directory contains the core API server implementation for fjcloud, organized into modules for HTTP routing, authentication, billing/invoicing, infrastructure provisioning across multiple cloud providers, data access repositories, and integrations with external services like Stripe, email, and DNS providers. |
| tests | The tests directory contains property-based tests for tenant isolation and shared testing utilities including mock implementations for Flapjack proxy and Stripe webhooks, along with helpers for index and storage route validation. |

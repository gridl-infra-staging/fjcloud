       expected API: https://api.staging.flapjack.foo
       observed API: https://api.flapjack.foo

            API_BASE_URL=https://api.staging.flapjack.foo -> trigger redeploy.
       Note: the env var is read via $env/dynamic/private at
             runtime (Cloudflare Pages Functions env), NOT a
             build-time PUBLIC_* var. See web/src/lib/config.ts.

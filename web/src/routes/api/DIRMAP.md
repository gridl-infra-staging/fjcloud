## api

| File | Summary |
| --- | --- |

| Directory | Summary |
| --- | --- |
| pricing | The pricing directory contains API endpoints for pricing-related functionality, with a compare subdirectory that provides server-side logic for comparing pricing options or plans. |
| search | This is a SvelteKit server route handler that processes API search requests, allowing clients to query and filter resources by dynamically specified entity types or resource names. |
| stripe | The stripe directory contains SvelteKit server routes for Stripe integration, including a publishable-key endpoint that serves public Stripe configuration to the frontend client. |

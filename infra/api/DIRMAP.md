| src | The src directory contains the Flapjack Cloud API service, an Axum-based HTTP server implementing multi-tenant billing and infrastructure management with integrated authentication, Stripe payments, cloud provisioning across AWS/GCP/OCI/Hetzner, DNS management, and email/webhook services. |
